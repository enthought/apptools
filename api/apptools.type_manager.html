class apptools.type_manager.abstract_type_system.AbstractTypeSystem[source]¶

Bases: traits.has_traits.HasTraits

The abstract base class for type systems.

A type system is responsible for:-

  1. Determining whether an object is of a particular type.
  2. Determining the MRO of a type.

See ‘PythonTypeSystem’ for an implementation with standard Python semantics.

apptools.type_manager.adapter_factory module¶

An adapter factory.

class apptools.type_manager.adapter_factory.AdapterFactory[source]¶

Bases: apptools.type_manager.abstract_adapter_factory.AbstractAdapterFactory

An adapter factory.

This implementation provides the common case where the factory adapts exactly one type of object to exactly one target type using exactly one adapter.

This class requires the adapter class to have an ‘adaptee’ trait. The default ‘Adapter’ class provides exactly that.

Module contents¶
